Backup Manager

Free Trial Version

USB and SSD Flash memory friendly

Backup Manager provides flash memory friendly backups to your pendrive or ssd drive.

A separate cli tool is available. check back this page for its release.

Key Features

  • Flash memory friendly
  • git like block based backups
  • file blocks are only written if changed
  • atomic file writes
  • NOTE: Backups are tied to configuration file contents (backup-tool-config.json), without the configuration file used while creating the backup, the backup cannot be extracted. We recommend backup backup-tool-config.json at a separate location than your backups.
